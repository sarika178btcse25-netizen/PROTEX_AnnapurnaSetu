# PROTEX_AnnapurnaSetu
Annapurna Setu is a real-time web platform designed to reduce food waste by connecting food donors, NGOs, and volunteer runners into a unified logistics network. It enables surplus food from restaurants, hotels, weddings, parties, and other events—essentially anywhere food is being wasted—to be redistributed efficiently to people in need before it expires.

The platform uses a priority-based sorting system to highlight the most time-sensitive food items, ensuring urgent deliveries are handled first. NGOs can browse and claim available food, while volunteer runners manage pickups and deliveries with the help of live tracking and location-based coordination. A verification mechanism ensures that each delivery is successfully completed.

To encourage active participation, the system incorporates a gamified reward model where volunteers earn points for completed deliveries, which can be redeemed through sponsor-backed incentives.

Annapurna Setu aims to build a scalable, technology-driven solution for hunger relief by making food redistribution faster, smarter, and more accessible across communities.
